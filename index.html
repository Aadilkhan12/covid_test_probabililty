About

The website is created by Aadil, 3rd year Bachelor in Computer Application (BCA) from Vivekananda Institute of Professional Studies (VIPS).
It uses India's Covid-19 dataset from Kaggle to train the data model and use it to predict patient's probability to get virus via personal records.
It uses python flask framework for website development and bootstrap for interactive UI design.
